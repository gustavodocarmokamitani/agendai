📅 Sistema de Agendamento
Este projeto é um sistema de agendamento front-end desenvolvido em React com TypeScript, que se conecta ao repositório BackendFoundation, responsável pelo back-end. O objetivo do sistema é fornecer uma interface robusta e intuitiva para gerenciar agendamentos de forma eficiente e prática.

🚀 Tecnologias utilizadas
React (com Hooks e Context API)
TypeScript
Axios - Para requisições HTTP
React Router - Para navegação entre páginas
Styled Components - Para estilização de componentes
Bootstrap - Framework CSS para layout responsivo
